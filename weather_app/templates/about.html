RESTful Weather Web Application

This is a RESTFUL API application done with Python and Django, containing weather data from the "area_clean_data.csv" csv file, to allow weather man to query data from the database as and when they need.

How to run the application:

  1. Navigate to this file: cd weather_app
  2. Create virtual environment: python3 -m venv myvenv
  3. Navigate to virtual environment: cd ../myvenv/bin
  4. Activate virtual environment: source activate
  5. Install a new application : pip3 install schedule
  6. Install django : pip3 install django ‘or’ python3 -m pip install django
  7. Install REST Framework : pip3 install djangorestframework
  8. Navigate to this file: cd weather_project
  9. Populate database with csv file: python3 ./scripts/loadRecord.py
  10. python3 manage.py makemigrations
  11. python3 manage.py migrate
  12. Run server: python3 manage.py runserver
  13. open http://127.0.0.1:8000/
